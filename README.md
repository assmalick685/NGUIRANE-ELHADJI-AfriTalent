# AfriTalent
Projet fil rouge — Plateforme de mise en relation entre freelances africains et clients.
# Auteur : EL HADJI NGUIRANE
# Promotion : L1 IAGE — ISI
# Description
AfriTalent est une plateforme numérique qui met en relation les freelances africains et les entreprises à la recherche de talents qualifiés. Elle permet aux professionnels de présenter leurs compétences, de trouver des missions et de développer leur carrière. Son objectif est de valoriser les talents africains et de faciliter les collaborations à l'échelle internationale.
# Technologies utilisées
HTML5, CSS3, JavaScript, BOOTSTRAP 5, Git et GitHub
# Fonctionnalités principales
1_ Mode sombre / clair,
2_ Bouton : Retour en haut,
3_ Navbar au défilement,
4_ Compteurs animés,

7_ Validation du formulaire de contact,
